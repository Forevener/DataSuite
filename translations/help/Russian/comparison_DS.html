Сравнительный анализ зависимых выборок

На данной странице представлены следующие статистические критерии:

Две выборки Несколько выборок
Параметрические t-критерий Стьюдента Анализ повторяющихся наблюдений
Непараметрические W-критерий Уилкоксона
критерий знаков		 Q-критерий Фридмана

Для проведения такого типа анализа таблица данных должна быть организована следующим образом: по каждому испытуемому сначала перечисляются все переменные первого замера, затем в том же порядке все испытуемые второго замера, и так далее. Таким образом, общее количество столбцов должно соответствовать количеству переменных, умноженному на количество замеров. Такой тип таблиц называется "широким". Никаких иных способов для разделения зависимых выборок не требуется. Это позволяет сравнивать любое количество переменных по любому количеству замеров максимально удобным способом.

При обнаружении нечисловых переменных анализ отменяется. Если наличие нечисловых переменных необходимо для разделения независимых выборок в комплексном исследовании, то сначала необходимо отфильтровать нужную выборку на странице загрузки и фильтрации данных, а затем исключить группирующую переменную из анализа.

В процессе расчёта критериев для сравнения нескольких выборок таблица данных преобразуется в "длинный" формат автоматически. Также "длинный" формат необходим при построении графиков.

Расчёт критерия знаков осуществляется с помощью пакета DescTools.

Все виды анализа на данной странице позволяют устанавливать силу и значимость различий между замерами/выборками по каждой переменной. В итоговые таблицы также добавляются значения центральной тенденции по каждой переменной для удобства сравнения замеров/выборок.

Выбор параметрического или непараметрического способа анализа зависит от размера и распределения выборки.

Столбцы основной выходной таблицы:

  • Центральная тенденция по каждому замеру/выборке
  • Выходное значение критерия (результат расчёта формулы)
  • Вероятность статистической ошибки (p-уровень)
  • Вывод о наличии различий

Строками основной выходной таблицы являются все числовые переменные.

Вывод о наличии различий и выделение p-уровня полужирным начертанием происходит в соответствии со значением, установленным в настройках приложения.

В случае сравнения нескольких выборок кроме результатов сравения всех замеров/выборок строятся таблицы попарных сравнений:

  • По критерию Стьюдента при применении анализа повторяющихся наблюдений
  • По критерию Уилкоксона при применении критерия Фридмана

Структура таблиц попарных сравнений: по горизонтальной и вертикальной оси перечислены выборки, в каждой ячейке указан скорректированный p-уровень при сравнении выборок из соответствующего столбца и строки.

Построение скрипичных графиков различий осуществляется с помощью пакета ggplot2. В графиках данного типа по горизонтальной оси откладывается замер/выборка, по вертикальной - значения переменных. Ширина фигуры показывает долю наблюдений/испытуемых, получивших данное значение. Технически, скрипичный график является графиком плотности распределения - повёрнутым на 90 градусов и отражённым по вертикальной оси.

На графики также накладываются следующие значения:

  • Среднее при параметрических видах анализа
  • Медиана, нижний и верхний квартили при непараметрических видах анализа
