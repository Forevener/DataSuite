Анализ распределения

На данной странице представлено три вида анализа:

  • Таблицы частот встречаемости
  • Гистограммы распределения
  • Проверка нормальности распределения по критерию Шапиро-Уилка

Расчёт таблиц частот встречаемости осуществляется при помощи пакета questionr. Данный тип анализа позволяет узнать, сколько раз в выборке наблюдалось каждое значения каждой переменной.

Структура таблицы частот:

  • Значение переменной
  • Количество наблюдений/испытуемых, имеющих данное значение
  • Процент наблюдений/испытуемых, имеющих данное значение, от общего числа

Построение графиков распределения осуществляется при помощи пакета ggplot2. Это простая визуализация таблиц частот встречамости.

При расчёт критерия Шапиро-Уилка из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений. Данный тип анализа позволяет узнать, является ли нормальным распределение каждой переменной.

Структура таблицы проверки нормальности:

  • Выходное значение критерия Шапиро-Уилка (результат расчёта формулы)
  • Вероятность статистической ошибки (p-уровень)
  • Вывод о нормальности распределения

Вывод о нормальности распределения и выделение p-уровня полужирным начертанием происходит в соответствии со значением, установленным в настройках приложения.
