Корреляционный анализ

На данной странице представлены следующие статистические критерии:

Тип Критерий
Параметрический r-критерий Пирсона
Непараметрический rho-критерий Спирмена
tau-критерий Кендалла

При запуске любого вида анализа на этой странице из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений.

Все виды анализа на данной странице позволяют устанавливать силу и значимость взаимосвязей между переменными.

Выбор параметрического или непараметрического способа анализа зависит от размера и распределения выборки.

Структура выходной таблицы коэффициентов корреляции: по горизонтальной и вертикальной оси перечислены переменные, выбранные в соответствующих полях, в каждой ячейке указан коэффициент при вычислении корреляции переменных из соответствующего столбца и строки.

Структура выходной таблицы p-уровней: по горизонтальной и вертикальной оси перечислены переменные, выбранные в соответствующих полях, в каждой ячейке указан p-уровень при вычислении корреляции переменных из соответствующего столбца и строки.

Выделение коэффициента корреляции и p-уровня полужирным начертанием происходит в соответствии со значением, установленным в настройках приложения.

Построение графиков разброса осуществляется с помощью пакета ggplot2. Для каждой корреляции (то есть - пары переменных) строится отдельный график. В графиках данного типа по горизонтальной оси откладывается первая переменная, по вертикальной - вторая переменная. Затем все наблюдения/испытуемые наносятся на график на основе значений осевых переменных.

На графики также накладывается кривая локальной регрессии с коридором погрешности.
