Кластерный анализ

На данной странице представлены статистические критерии для осуществления кластерного анализа.

При запуске любого вида анализа на этой странице из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений. Теоретически, выборка для кластерного анализа должна удовлетворять требованиям параметрических критериев.

Кластерный анализ позволяет выявлять относительно устойчивые группы наблюдений/испытуемых, сходные между собой по тем переменным, которые участвовали в анализе. Вывод о сходстве делается на основе евклидового расстояния между наблюдениями/испытуемыми в многомерном пространстве, где измерениями выступают переменные.

Подбор количества кластеров позволяет установить оптимальное количество кластеров для данной выборки.

Пакет NbClust рассчитывает значения 30 критериев качества кластеризации и делает вывод об оптимальном количесве, опираясь на максимальные значения качества по большинству критериев.

Некоторые из этих критериев и некоторые дополнительные выражаются графически:

  1. Дендрограмма (график иерархической кластеризации) - показывает евклидовое расстояние, на котором находились все испытуемые/наблюдения, и в каком порядке они объединялись бы в кластеры. Кластеризация для этого графика проводится по методу полной привязки.
  2. Дерево кластеризации (пакет clustree) - показывает соотношение и процесс разбивки кластеров при различных количествах кластеров в процессе кластеризации по методу K-средних.
  3. График осыпи кластеров (пакет factoextra) - показывает внутри-кластерные суммы квадратов при различных количествах кластеров в процессе кластеризации по методу K-средних. Теоретически, выбрать нужно то количество кластеров, которое находится на "переломе" графика.
  4. График силуэтов кластеров (пакет factoextra) - показывает среднюю ширину силуэта кластеров при различных количествах кластеров в процессе кластеризации по методу K-средних. Теоретически, выбрать нужно то количество кластеров, для которого средняя ширина силуэтов является максимальной.
  5. График индексов качества кластеризации (пакет ggplot2) - показывает, какое количество кластеров имеют максимальное значение качества по какому количество критериев пакета NbClust.

Для проведения кластерного анализа необходимо указать количество кластеров. Анализ проводится методом K-средних, и позволяет получить следующие результаты:

  1. Таблица состава кластеров показывает, сколько наблюдений/испытуемых вошло в каждый кластер.
  2. Таблица средних показывает, какие средние значения по каждой переменной получены в каждом кластере.
  3. Индексы качества кластеризации:
    • Меж-кластерное расстояние (среднее расстояние между объектами разных кластеров - чем больше, тем лучше)
    • Внутри-кластерное расстояние (среднее расстояние между объектами одного кластера - чем меньше, тем лучше)
    • Средняя ширина силуэта (степень отличия расстояния между объектами своего кластера и расстояния до объектов другого кластера - измеряется от -1 до +1. Чем больше, тем лучше)
    • Индексы Данна (мера компактности кластеров и удалённости их друг от друга - чем больше, тем лучше)

Самым главным результатом кластеризации является таблица данных с указанием принадлежности каждой строки к кластеру. Эту таблицу можно скачать по ссылке в конце блока основных результатов.

Графики кластеризации:

  1. График положения объектов (пакет factoextra) - показывает положение и принадлежность каждого объекта по двум основным осям, полученным из переменных с помощью анализа главных компонентов
  2. График силуэтов (пакет factoextra) - показывает силуэт каждого объекта в каждом кластере
