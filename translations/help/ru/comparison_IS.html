Сравнительный анализ независимых выборок

На данной странице представлены следующие статистические критерии:

Две выборки Несколько выборок
Параметрические t-критерий Стьюдента F-критерий Уэлча
Непараметрические U-критерий Манна-Уитни-Уилкоксона
D-критерий Колмогорова-Смирнова
Z-критерий Уалда-Уолфовитца		 H-критерий Краскела-Уоллиса

Для проведения такого типа анализа в таблице данных должна присутствовать группирующая переменная, фиксирующая отношение каждого наблюдения/испытуемого к какой-либо категории. Такой тип таблиц называется "длинным". Никаких иных способов для разделения независимых выборок не требуется.

При загрузке или фильтрации таблицы исходных данных все переменные, имеющие только два возможных значения, вносятся в список потенциальных независимых переменных для сравнения двух независимых выборок. Если требуется провести сравнение двух зависимых выборок по переменной, в которой содержится большее количество возможных значений, то предварительно необходимо отфильтровать лишние значения на странице ввода/фильтрации данных.

В список потенциальных независимых переменных для сравнения нескольких независимых выборок вносятся все переменные из таблицы данных (кроме переменных с нулевым разбросом значений). Числовые независимые переменные в процессе анализа будут преобразованы в факторные переменные.

При запуске любого вида анализа на этой странице из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений.

Расчёт Z-критерия Уалда-Уолфовитца осуществляется с помощью пакета DescTools.

Все виды анализа на данной странице позволяют устанавливать силу и значимость различий между выборками по каждой переменной. В итоговые таблицы также добавляются значения центральной тенденции по каждой переменной для удобства сравнения выборок.

Выбор параметрического или непараметрического способа анализа зависит от размера и распределения выборки.

Столбцы основной выходной таблицы:

  • Центральная тенденция по каждой выборке
  • Выходное значение критерия (результат расчёта формулы)
  • Вероятность статистической ошибки (p-уровень)
  • Вывод о наличии различий

Строками основной выходной таблицы являются все числовые переменные.

Вывод о наличии различий и выделение p-уровня полужирным начертанием происходит в соответствии со значением, установленным в настройках приложения.

В случае сравнения нескольких выборок кроме результатов сравения всех выборок строятся таблицы попарных сравнений:

  • По критерию Стьюдента при применении критерия Уэлча
  • По критерию Манна-Уитни-Уилкоксона при применении критерия Краскела-Уоллиса

Структура таблиц попарных сравнений: по горизонтальной и вертикальной оси перечислены выборки, в каждой ячейке указан скорректированный p-уровень при сравнении выборок из соответствующего столбца и строки.

Построение скрипичных графиков различий осуществляется с помощью пакета ggplot2. В графиках данного типа по горизонтальной оси откладывается выборка, по вертикальной - значения переменных. Ширина фигуры показывает долю наблюдений/испытуемых, получивших данное значение. Технически, скрипичный график является графиком плотности распределения - повёрнутым на 90 градусов и отражённым по вертикальной оси.

На графики также накладываются следующие значения:

  • Среднее при параметрических видах анализа
  • Медиана, нижний и верхний квартили при непараметрических видах анализа
