Регрессионный анализ

На данной странице представлены инструменты для осуществления регрессионного анализа.

Теоретически, выборка для регрессионного анализа должна удовлетворять требованиям параметрических критериев.

Регрессионный анализ позволяет выявлять независимые переменные, влияющие на зависимую переменную, определить степень этого влияния и его статистическую значимость. Следует учесть, что вывод о том, что зависимая переменная определяется независимыми, нельзя делать на основе одного только регрессионного анализа.

Условно, регрессионный анализ позволяет установить "формулу" зависимой переменной - в идеале, её значение будет равняться сумме значений независимых переменных, умноженных на их коэффициенты.

В список потенциальных независимых переменных вносятся все переменные из таблицы данных (кроме переменных с нулевым разбросом значений). В список потенциальных зависимых переменных вносятся все числовые переменные из таблицы данных (кроме переменных с нулевым разбросом значений).

Перед началом анализа необходимо выбрать зависимые и независимые переменные. Если зависимых переменных выбрано несколько, анализ будет проводиться для каждой из них отдельно.

Под моделью понимается тот набор независимых переменных, влияние которых должен установить анализ.

Обобщённые линейные модели

Линейные модели предназначены для выявления линейных закономерностей.

Оптимизация модели с помощью пакета leaps позволяет рассчитать индекс качества для каждой возможной модели и, таким образом, обнаружить модель с максимальным качеством.

Основным результатом является график моделей. По вертикальной оси отмечается качество (скорректированный r2), по горизонтальной - все независимые переменные (в случае с нечисловыми переменными указываются их конкретные значения). Переменные, присутствующие в модели, отмечаются чёрными блоками, отсутствующие - белыми. Таким образом, на верхней строке графика чёрными блоками отмечены переменные из наиболее качественной модели.

Следует учесть, что оптимизация модели не во всех случаях является желательной процедурой, поэтому проводить её рекомендуется только в том случае, если это явным образом описано в задачах исследования.

Основным результатом анализа является таблица коэффициентов. Строками таблицы выступают независимые переменные, а также неучтённое воздействие - intercept. Столбцами таблицы выступают:

  • r - коэффициент регрессии
  • Стандартная ошибка - стандартное отклонение коэффициента в выборке
  • t - коэффициент, делённый на стандартную ошибку
  • p - вероятность статистической ошибки (выделяется полужирным начертанием в соответствии с настройкой приложения)

Также для зависимой переменной рассчитываются три индекса качества модели. При сравнении разных моделей предпочтение должно отдаваться моделям с наименьшими значениями критериев Акаике и Байеса, и наибольшими значениями по логарифмическому правдоподобию.

График регрессии (пакет sjPlot) отображает значения коэффициентов по каждой независимой переменной, а также отдельно выделяет знак коэффициента цветом.
