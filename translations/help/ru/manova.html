Многофакторный дисперсионный анализ

На данной странице представлен многофакторный дисперсионный анализ.

Для проведения такого типа анализа в таблице данных должны присутствовать группирующие переменные, фиксирующие отношение каждого наблюдения/испытуемого к каким-либо категориям. Такой тип таблиц называется "длинным". Никаких иных способов для разделения независимых выборок не требуется.

В список потенциальных независимых переменных вносятся все переменные из таблицы данных (кроме переменных с нулевым разбросом значений). Числовые независимые переменные в процессе анализа будут преобразованы в факторные переменные.

При запуске анализа на этой странице из списка зависимых переменных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений.

Данный анализ позволяют устанавливать силу и значимость различий между под-выборками, выделенными по нескольким критериям, по каждой переменной. Например, если требуется выявить различия между разными видами черепах с учётом их пола, то в исследовании будет фигурировать две независимых переменных - вид и пол. По этим двум переменным будет составлены под-выборки, имеющие все возможные комбинации вида и пола (следует заметить, что многофакторный дисперсионный анализ следует проводить тогда, когда в данных действительно присутствуют все возможные комбинации в достаточном количестве).

Теоретически, выборка должна удовлетворять требованиям параметрических критериев.

Столбцы основной выходной таблицы:

  1. Степени свободы
  2. Сумма квадратов
  3. Среднее квадратов
  4. F - выходное значение
  5. p - вероятность статистической ошибки

Строками основной выходной таблицы являются разные комбинации независимых переменных и остаток (расхождения между предполагаемыми и наблюдаемыми данными).

Вывод о наличии различий и выделение p-уровня полужирным начертанием происходит в соответствии со значением, установленным в настройках приложения.

Структура таблиц попарных сравнений: в строках указаны пары под-выборок, в столбцах указаны следующие значения:

  1. Разница средних
  2. Нижняя граница интервала распределения
  3. Верхняя граница интервала распределения
  4. Скорректированный для множества замеров p-уровнь (выделяется полужирным начертанием в соответствии с настройками приложения)

Графики взаимодействия - по горизонтальной оси откладываются независимые переменные, по вертикальной - значения переменной. По каждой переменной также строится ось, на которой отмечаются средние значения для каждого уровня. Таким образом, можно обнаружить независимые переменные, имеющие больший эффект.
