Страница загрузки и фильтрации данных


Данная страница позволяет загрузить файлы следующих форматов в приложение:

  • *.csv
  • *.dta
  • *.ods
  • *.por
  • *.sas
  • *.sav
  • *.tsv
  • *.xls
  • *.xlsx
  • *.xpt
  • *.zsav

Файл данных должен соответствовать следующим требованиям:

  • Строки таблицы представляют собой случаи/испытуемых
  • Столбцы таблицы представляют собой переменные
  • В первой строке (и только в ней) должны содержаться названия переменных

Пустые столбцы и пустые строки будут удалены из таблицы автоматически.


С помощью инструментов поиска либо фильтрации в таблице можно выбрать случаи (испытуемых) для обработки во всех дальнейших анализах.

Некоторые переменные можно исключить из анализа, если они являются заведомо лишними, либо позволяют отбирать случаи/испытуемых, но не должны сами участвовать в анализе. Это может потребоваться при сравнении сгруппированных зависимых выборок.

Пока файл не загружен, анализы на других страницах будут недоступны.

При переходе из данной страницы интерфейс приложения заполняется каждый раз заново, и результаты предыдущих анализов будут удалены.
