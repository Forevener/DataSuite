Факторный анализ

На данной странице представлены инструменты для осуществления факторного анализа и анализа главных компонентов из пакета psych.

При запуске любого вида анализа на этой странице из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений. Теоретически, выборка для факторного анализа должна удовлетворять требованиям параметрических критериев.

Факторный анализ позволяет осуществлять "сжатие" данных, выделяя факторы из связанных между собой переменных. Связанность переменных можно интерпретировать различным образом - например, переменные описывают одно и то же явление, или переменными управляет некоторая неявная структура. Таким образом, факторный анализ может помочь в оптимизации модели данных путём отсечения лишних переменных, или в поиске неких скрытых закономерностей или явлений.

Перед проведением факторного анализа неоходимо выбрать метод факторизации и метод вращения матрицы нагрузок.

Подбор количества факторов позволяет установить оптимальное количество факторов для данной выборки.

Пакет psycho рассчитывает значения 7 критериев качества факторизации и делает вывод об оптимальном количесве, опираясь на максимальные значения качества по большинству критериев.

Некоторые из этих критериев и некоторые дополнительные выражаются следующим образом:

  1. График каменистой осыпи (пакет ggplot2) - показывает собственные значения факторов при различных количествах факторов. Теоретически, выбрать нужно то количество факторов, которое находится на "переломе" графика.
  2. Метод параллельного анализа - строится на основе графика каменистой осыпи, с добавлением симулированных данных. Теоретически, выбрать нужно то количество факторов, собственные значения которых превосходят собственные значения симулированных факторов.
  3. Критерий Кайзера - рекомендует выбирать то количество факторов, собственные значения которых превосходят единицу. На графике каменистой осыпи единица отмечена чёрной линией.

Для проведения факторного анализа необходимо указать количество факторов, и отметить, если требуется, нормализацию вращения матрицы (не применяется для вращения Эквамакс).

Основным результатом анализа является таблица нагрузок, в которой отмечается нагрузка каждой переменной в каждом факторе. В этой таблице нагрузки, меньшие по модулю настройки отсечения, не показываются. Нагрузки, большие по модулю настройки выделения, отмечаются полужирным начертанием.

Дополнительные результаты:

  1. Подробная таблица нагрузок показывает все нагрузки без исключения, но выделяет те, которые превосходят по модулю настройку выделения. Кроме этого, в таблицу добавляются значения:
    • Уникальности (та часть дисперсии фактора, которая не пересекается с дисперсиями других факторов)
    • Сложности (Индекс Хоффмана - мера нагруженности переменной по всем факторам)
    • Общности (та часть дисперсии фактора, которая пересекается с дисперсиями других факторов)
  2. Таблица сведений о факторах демонстрирует четыре показателя по каждому фактору:
    • Собственное значение
    • Объясняемая дисперсия
    • Доля общей дисперсии
    • Доля объясняемой дисперсии
  3. Индексы качества факторизации:
    • Корень квадратов остатков (чем ближе к нулю - тем лучше)
    • Корень среднего квадрата ошибки аппроксимации (желательно 0.05 и меньше)
    • Индекс Такера-Льюиса (желательно 0.95 и больше)

Графики факторизации:

  1. График отношения переменных (пакет psych) - показывает к какому компоненту или фактору относится каждая переменная, с указанием нагрузки и её направлением (отрицательные нагрузки выделяются красным цветом и пунктирной линией)
  2. График нагрузок (пакет ggplot2) - показывает нагрузку каждого фактора по каждой переменной на графике радарного типа. Нагрузки, меньшие по модулю настройки отсечения, не показываются.
