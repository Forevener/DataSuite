Анализ надёжности

При запуске анализа на этой странице из таблицы данных автоматически удаляются любые нечисловые переменные и числовые переменные с нулевым разбросом значений.

Анализ надёжности позволяет установить согласованность элементов - например, вопросов одной шкалы в психологическом опроснике. Для данного анализа используется пакет psych.

Для проведения анализа нужно выбрать переменные (пункты опросника или шкалы), согласованность которых необходимо установить, и дополнительно указать обратные переменные.

Первая основная таблица позволяет установить следующие коэффиценты:

  • Альфа Кронбаха
  • Лямбда-6 Гуттмана
  • Омега МакДональда
  • Показатель композитной надёжности

Вторая основная таблица (таблица отбросов) демонстрирует как изменятся показатели Альфа и Лямбда-6, если будут исключены соответствующие переменные.

Первая дополнительная таблица (расширенная таблица отбросов) демонстрирует изменения других характеристик опросника при отбросе переменных. Общие показатели указаны в первой строке, затем перечисляются все переменные.

Вторая дополнительная таблица (статистика элементов) демонстрирует количество элементов, различные показатели корреляции каждого элемента с общим результатом, а также изменения корреляции при отбросе переменной.
