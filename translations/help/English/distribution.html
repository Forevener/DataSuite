Distribution analysis

This page provides three types of analysis:

  • Frequency tables
  • Distribution Histograms
  • Verification of the normality of the distribution by Shapiro-Wilk's test

The calculation of frequency tables is carried out using the questionr package. This type of analysis allows you to find out how many times each value of each variable was observed in the sample.

Frequency table structure:

  • Variable Value
  • Number of observations / subjects having this value
  • Percentage of observations / subjects having this value

Distribution graphs are built using the ggplot2 package. This is a simple visualization of frequency tables.

When calculating the Shapiro-Wilk criterion, any non-numeric variables and numerical variables with zero variance ​​are automatically deleted from the data table. This type of analysis allows you to find out if the distribution of each variable is normal.

Normality check table structure:

  • The output value of the Shapiro-Wilk test (the result of calculating the formula)
  • Probability of statistical error (p-level)
  • Distribution inference

The conclusion about the normality of the distribution and the highlighting of the p-level in bold happens in accordance with the value set in the application settings.
