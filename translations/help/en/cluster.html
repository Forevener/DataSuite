Cluster analysis

This page provides statistical criteria for conducting cluster analysis.

When you run any kind of analysis on this page, any non-numeric variables and numeric variables with zero variance are automatically removed from the data table. Theoretically, the sample for cluster analysis should satisfy the requirements of parametric criteria.

Cluster analysis allows you to identify relatively stable groups of observations / subjects, similar to each other in terms of variables participating in the analysis. The conclusion about the similarity is made on the basis of the Euclidean distance between the observations/subjects in a multidimensional space, where the variables are dimensions.

Selecting the number of clusters allows to find the optimal number of clusters for the sample.

The NbClust package calculates the values of 30 criteria for the quality of clustering and discovers the optimal quantity based on the maximum quality values ​​for most criteria.

Some of these criteria and some additional ones are expressed graphically:

  1. Dendrogram (hierarchical clustering graph) - shows the Euclidean distance at which all subjects / observations were located, and in what order they would be combined into clusters. Clustering for this graph is done using the complete linkage method.
  2. Сlustering tree (clustree package) - shows the ratio and process of breaking up clusters with different numbers of clusters during the clustering process using the K-means method.
  3. Cluster scree graph (factoextra package) - shows the intra-cluster sums of squares for various numbers of clusters during the clustering process using the K-means method. Theoretically, you need to choose the number of clusters that is on the "knee" of the chart.
  4. Cluster silhouette graph (factoextra package) - shows the average cluster silhouette width for various numbers of clusters during the clustering process using the K-means method. Theoretically, you need to select the number of clusters for which the average width of the silhouettes is maximal.
  5. Clusterization Quality Index Graph (ggplot2 package) - shows which clustering models have the maximum support according to criteria of the NbClust package.

To conduct cluster analysis, you must specify the number of clusters. The analysis is carried out using the K-means method, and allows to obtain the following results:

  1. The cluster composition table shows how many observations / subjects were included in each cluster.
  2. The average table shows what average values ​​for each variable in each cluster.
  3. Clustering Quality Indices:
    • Intercluster distance (the average distance between objects of different clusters - the larger the better)
    • Intracluster distance (the average distance between objects of the same cluster - the smaller the better)
    • The average width of the silhouette (the degree of difference between the distance between the objects of its cluster and the distance to the objects of another cluster - measured from -1 to +1. The larger the better)
    • Dunn indices (a measure of cluster compactness and their distance from each other - the more the better)

The most important result of clustering is a data table indicating each row belongs to the cluster. This table can be downloaded from the link at the end of the main results block.

Clustering Graphs:

  1. Graph of the position of objects (factoextra package) - shows the position and belonging of each object along two main axes obtained from variables by analyzing the main components
  2. Silhouette graph (factoextra package) - shows the silhouette of each object in each cluster
