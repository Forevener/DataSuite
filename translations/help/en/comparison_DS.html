Comparison of dependent samples

The following statistical criteria are presented on this page:

Two samples Multiple samples
Parametric Student t-test Analysis of repeated observations
Nonparametric Wilcoxon W-test
Sign test		 Friedman’s Q-test

To carry out this type of analysis, the data table should be organized as follows: for each subject, all the variables of the first measurement are listed first, then in the same order all the subjects of the second measurement, and so on. Thus,the total number of columns should correspond to the number of variables times the number of measurements. This type of table is called "wide." No other methods are required to separate dependent samples. This allows you to compare any number of variables for any number of measurements in the most convenient way.

If non-numeric variables are detected, the analysis is canceled. If the presence of non-numeric variables is necessary for separating independent samples in a complex study, you must first filter the desired selection on the data uploading and filtering page, and then exclude the grouping variable from the analysis.

In the process of calculating the criteria for comparing multiple samples, the data table is automatically converted to the "long" format. Also, a "long" format is needed when plotting.

Sign test is calculated using the DescTools package.

All types of analysis on this page allow you to find the strength and significance of the differences between measurements / samples for each variable. Values ​​of the central tendency for each variable are also added to the summary tables for easy comparison of measurements / samples.

The choice of a parametric or non-parametric analysis method depends on the size and distribution of the sample, and also criteria-specific requirements.

Columns of the main output table:

  • Central tendency for each measurement / sample
  • The output value of the test (the result of calculating the formula)
  • Probability of statistical error (p-level)
  • Difference inference

The rows of the main output table are all numeric variables of the data table.

The conclusion about the presence of differences and the highlighting of the p-level in bold is happens in accordance with the value set in the application settings.

In case of comparing several samples, in addition to the results of comparing all measurements / samples, tables of pairwise comparisons are built:

  • By Student's test when applying the analysis of repeated observations
  • By the Wilcoxon's test when applying the Friedman test

Structure of pairwise comparison tables: samples are listed along the horizontal and vertical axis, in each cell the adjusted p-level is indicated when comparing samples from the corresponding column and row.

The construction of violin plots of differences is carried out using the ggplot2 package. In graphs of this type, measurement / sample is situated along the horizontal axis, and values ​​of variable along the vertical axis. The width of the figure shows the proportion of observations / subjects who received correspondent value. Technically, the violin plot is a distribution density plot - rotated 90 degrees and mirrored along the vertical axis.

The following values ​​are also superimposed on charts:

  • Average for parametric tests
  • Median, lower and upper quartiles for nonparametric tests
