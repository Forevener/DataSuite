Data upload and filter page


This page allows you to upload files of the following formats to the application:

  • *.csv
  • *.dta
  • *.ods
  • *.por
  • *.sas
  • *.sav
  • *.tsv
  • *.xls
  • *.xlsx
  • *.xpt
  • *.zsav

The data file must meet the following requirements:

  • Table rows represent cases / subjects
  • Table columns represent variables
  • The first row (and only it) should contain the names of variables

Empty columns and empty rows will be removed from the table automatically.


Using the search or filtering tools in the table, you can select cases (subjects) for processing in all further analyzes.

Some variables can be excluded from the analysis if they are obviously redundant, or they allow you to select cases / subjects, but should not be involved in the analysis themselves. This may be required when comparing grouped dependent samples.

Until the file is uploaded, analyzes on other pages will not be available.

When you go from this page, the application interface is re-filled each time, and the results of previous analyzes will be deleted.
