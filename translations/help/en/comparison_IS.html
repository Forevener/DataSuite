Comparisons of independent samples

The following statistical criteria are presented on this page:

Two samples Multiple samples
Parametric Student t-test Welch F-test
Nonparametric Mann-Whitney-Wilcoxon U-test
Kolmogorov-Smirnov D-test
Wald-Wolfowitz Z-test		 Kruskal-Wallis H-test

To carry out this type of analysis, a grouping variable must be present in the data table that determines the category (sample) of each observation / subject. This type of table is called "long." No other way is needed to separate independent samples.

When loading or filtering the source data table, all variables that have only two possible values ​​are added to the list of potential independent variables for comparing two independent samples. If you want to compare two dependent samples by a variable that contains more possible values, then first you need to filter out the extra values ​​on the data upload and filtering page.

All variables from the data table are added into the list of potential independent variables for comparing several independent samples (except for variables with a zero variance). Numeric independent variables will be converted into factor variables during the analysis.

When you run any type of analysis on this page, any non-numeric variables and numerical variables with zero variance ​​are automatically removed from the data table.

The Wald-Wolfowitz Z-test is calculated using the DescTools package.

All types of analysis on this page allow you to find the strength and significance of the differences between the samples for each variable. Values ​​of the central tendency for each variable are also added to the summary tables for easy comparison of samples.

The choice of a parametric or non-parametric analysis method depends on the size and distribution of the sample (and also criteria-specific requirements).

Columns of the main output table:

  • Central tendency for each sample
  • The output value of the test (the result of calculating the formula)
  • Probability of statistical error (p-level)
  • Difference inference

The rows of the main output table are all numeric variables of the data table.

The conclusion about the presence of differences and the highlighting of the p-level in bold is happens in accordance with the value set in the application settings.

In the case of comparing several samples, in addition to the results of comparing all samples, tables of pairwise comparisons are constructed:

  • By the Student's test when applying the Welch test
  • By the Mann-Whitney-Wilcoxon's test when applying the Kruskal-Wallis test

Structure of pairwise comparison tables: samples are listed along the horizontal and vertical axis, in each cell the adjusted p-level is indicated when comparing samples from the corresponding column and row.

The construction of violin plots of differences is carried out using the ggplot2 package. In graphs of this type, the sample is laid out along the horizontal axis, and the values ​​of the variables along the vertical axis. The width of the figure shows the proportion of observations / subjects which received this value. Technically, the violin plot is a distribution density plot - rotated 90 degrees and mirrored along the vertical axis.

The following values ​​are also superimposed on charts:

  • Average for parametric analyzes
  • Median, lower and upper quartiles for nonparametric analyzes
