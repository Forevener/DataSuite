Correlation analysis

The following statistical criteria are presented on this page:

Type Criterion
Parametric Pearson's r-test
Nonparametric Spearman's rho test
Kendall's tau test

When you run any type of analysis on this page, any non-numeric variables and numerical variables with zero variance ​​are automatically deleted from the data table.

All types of analysis on this page allow you to find the strength and significance of the relationships between variables.

The choice of a parametric or non-parametric analysis method depends on the size and distribution of the sample (and also criteria-specific requirements).

Structure of the output table of correlation coefficients: the horizontal and vertical axis lists the variables selected in the corresponding fields, each cell shows the coefficient when calculating the correlation of variables from the corresponding column and row.

The structure of the output table of p-levels: the horizontal and vertical axis lists the variables selected in the corresponding fields, each cell shows the p-level when calculating the correlation of variables from the corresponding column and row.

Highlighting of the correlation coefficient and p-level in bold is happens in accordance with the value set in the application settings.

Scatter plots are built using the ggplot2 package. For each correlation (that is, pairs of variables), a separate graph is constructed. In graphs of this type, the first variable is plotted on the horizontal axis, and the second variable on the vertical axis. Then all observations / subjects are plotted based on the values ​​of the axial variables.

A local regression curve with an error corridor is also superimposed on the graphs.
